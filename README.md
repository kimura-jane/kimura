# かなたけ×木村のおにぎり占い

おにぎりの具や気分から、あなたにぴったりのおにぎりと今日の運勢を占うシンプルな診断サイトです。

## 🔧 使い方

1. `index.html` をブラウザで開く  
2. 「スタート ▶︎」ボタンをクリック  
3. 質問に答えると結果ページに移動  
4. タイプ別のおにぎり ＋ 今日の運勢 ＋ ラッキー要素 が表示されます
